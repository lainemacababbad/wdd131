Site Plan

Site Name:

Pinoy Flavors

Site Purpose:

Pinoy Flavors is dedicated to exploring the diverse and vibrant dishes of the Philippines, offering insights into their origins, cultural significance, and variations across different islands.

Scenarios:

  • How can I recreate these Filipino recipes at home?
  • Are there any vegetarian or vegan options in Filipino cuisine?
  • What are some must-try Filipino dishes?
  • How easy is it to cook Filipino food?

Color Scheme:

I plan to incorporate the following colors in my website:

#0038A8 (Header)

This blue will be used for the headers.

#C8102E (Navigation and Footer)

Red will be used for the navigation and footer.

#EFE5D5 (Background)

This light color will be used as one of the background colors.

#FDD2C3 (Background)

This light pink will be used as another background color.

#FCD116 (Buttons)

This yellow will be used for buttons.

Typography:

The font that I will be using for the logo/site name is Poltawski Nowy.

The font that I will be using for the heading is Archivo Black. It is a bold display sans-serif font and definitely not intended for body text.

The font that I will be using for the body text is Poppins, which gives a modern and clean look with rounded edges, making it easy to read.
